Tools I use

Here contains the list of libraries and software I use for web development.

For this site...

SASS, a CSS preprocessor that empowers CSS with logical operators, mixins, nesting, and more, before compilation.

npm, a registry for JavaScript libraries I download and use.

Prettier, a JavaScript library used to prettify the code into a more readable way, depending on your configuration.

localhost, a library used to host a site in development.

VSCode, an editor with vast amount of extensions and flexibility, used for coding stuff.

Google, a search engine helps searching and "learning" for a solution.

For others...

Some from the previous topics, plus...

Node.js, a JavaScript runtime environment that I only use its file system and never give a fuck with its other features.

Svelte, a UI library used to create reactive components. Kinda like React, but easier to learn, code, and read. Plus, it's faster, since it compiles.

SvelteKit, a UI framework used to create many types of website like SSR, SSG, SPA, or MPA. It also can handle some back-end job with Node.js like file routing, or perform a fetch from database, etc.

Currently learning...

Vue, a framework for building UIs, like Svelte, but with a larger ecosystem and jobs ($$$), although slower, since it ships runtime, and harder to read and code.
