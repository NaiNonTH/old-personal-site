Chosen Emojis

How to use

Start by choosing some emojis from the list given. You can choose more than one.

To remove chosen emojis, click on the Selected emojis above the list.

Then, change the Output Length and the Repetition Chance:

  • Output Length defines how much it should output the set of emojis that will be output. It receives any whole number (but please, not too big).
  • Repetition Chance defines how less variated the set of output emojis should be. It receives a number ranging from 1 to 100.

After done filling in inputs, click "Random Spam!" for your output.

If you don't like the output, you can always click "Random Spam!" again.

Note

  • There are chances that some emojis might not appear in the output. Though, you can rerun the process again for the output that includes all emojis. A higher Repetition Chance can increase this behavior.
  • Due to technical limitation, I can't implement the emoji input as a textbox that you can type in.
